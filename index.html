FizzBuzz Checker

FizzBuzz Project Summary

What I Learned

While working on this project, I learned the following:

  • How to set up a TypeScript project using tsc to compile TypeScript into JavaScript.
  • The basics of TypeScript, including type annotations, functions, and methods.
  • How to create a simple FizzBuzz program to handle user input and display results dynamically.
  • Integrating TypeScript with HTML by generating JavaScript files in a dist folder.

Project Functionality

This project includes the following functionality:

  1. A text input where users can enter a number.
  2. A "Check" button that triggers the FizzBuzz logic.
  3. A result display area that shows:
    • "Fizz" for numbers divisible by 3.
    • "Buzz" for numbers divisible by 5.
    • "FizzBuzz" for numbers divisible by both 3 and 5.
    • The entered number for others.
